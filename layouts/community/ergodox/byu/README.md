## Layout

### Keymap 0: Basic layer
```
 *
 * ,--------------------------------------------------.           ,--------------------------------------------------.
 * |  =     |   1  |   2  |   3  |   4  |   5  |      |           |      |   6  |   7  |   8  |   9  |   0  |   -    |
 * |--------+------+------+------+------+-------------|           |------+------+------+------+------+------+--------|
 * | TAB    |   Q  |   W  |   E  |   R  |   T  |      |           |      |   Y  |   U  |   I  |   O  |   P  | LT3, \ |
 * |--------+------+------+------+------+------|      |           |      |------+------+------+------+------+--------|
 * |Cmd/Esc |   A  |   S  |   D  |LT3, F|   G  |------|           |------|   H  |   J  |   K  |   L  |   ;  |   '    |
 * |--------+------+------+------+------+------|      |           |      |------+------+------+------+------+--------|
 * | LShift |   Z  |   X  |   C  |   V  |   B  |      |           |      |   N  |   M  |   ,  |   .  |   /  | RShift |
 * `--------+------+------+------+------+-------------'           `-------------+------+------+------+------+--------'
 *   |LCtrl |LAlt/`|  [   |   ]  | L1   |                                       | Left | Down |  Up  | Right|  L2  |
 *   `----------------------------------'                                       `----------------------------------'
 *                                      ,--------------.       ,-------------.
 *                                      |  Cmd  |  `   |       | VolD | VolU |
 *                               ,------|-------|------|       |------+--------+------.
 *                               |      |       | Caps |       | Mute |        |      |
 *                               | Bcksp|  Del  |------|       |------|  Ent   |Space |
 *                               |      |       | Hyper|       | Meh  |        |      |
 *                               `---------------------'       `----------------------'
```
* Press-and-hold `f` to go to the movement layer
